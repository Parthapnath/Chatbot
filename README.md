# 💬 Interactive Chatbot (HTML/CSS/JS)

An interactive and responsive chatbot built using **HTML**, **CSS**, and **JavaScript**. It can handle user text inputs, display emojis, respond to questions, and even handle images — making it a fun and engaging UI project.

## 🌟 Features

- ✨ Real-time text-based interaction
- 🖼️ Image-based question and response support
- 😄 Emoji recognition and responses
- 📱 Responsive UI (works across devices)
- 🎨 Clean, customizable UI with CSS
